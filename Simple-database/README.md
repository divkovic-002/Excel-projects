# Tech Store - Sales & Warranty Tracker

## Description
This is an Excel sheet designed for tracking sales and managing product warranties for a retail store ("Tech Store"). The goal of this project is to clean up raw data, keep everything organized, and automate daily retail calculations for better readability.

## Features
* **Automated VAT & Warranty Lookup:** Uses reference tables (via VLOOKUP/XLOOKUP formulas) to automatically apply the correct dynamic VAT rate and warranty days based on the product category.
* **Real-Time Warranty Tracking:** Calculates the exact expiration date and automatically compares it with the current date.
* **Conditional Formatting:** Color-codes the warranty status automatically (Red for expired / *Prošla*, Green for active / *Traje*) and includes visual data bars for total amounts.
* **Clean & Readable Layout:** Structured with clear headers and a neat color-coded design for quick and easy navigation.

## Technologies Used
* Microsoft Excel (`.xlsx`)
* Excel Formulas (Lookup and Date functions)
* Conditional Formatting
